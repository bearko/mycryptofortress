# Human-in-the-Loop（HITL）論点

PR レビューで人間が判断する論点の一覧。実装 PR では該当する論点をチェックして理由を添える。

## 該当する論点

- [ ] **アセット権利**: MCH 公式ガイドラインに抵触しうる新規アセットの追加（コラボ・権利除外対象）。
- [ ] **ゲーム数値の大幅変更**: ヒーロー/敵のステータス、ダメージ係数、CE 蓄積速度などゲーム体験を変える数値の調整。
- [ ] **依存関係の大幅更新**: Phaser / Vite / TypeScript の major アップデート、新規ランタイム依存追加。
- [ ] **配布・デプロイ方針の変更**: GitHub Pages / Vercel など配信先の変更や、ビルド成果物のサイズが大幅増。
- [ ] **個人情報・トラッキング**: アナリティクスや SNS 連携の追加（このプロジェクトでは原則なし）。
- [ ] **破壊的 API / セーブデータ互換**: 将来追加予定のセーブデータ形式を変える変更。
- [ ] **不可逆操作**: 保護ブランチへの force push、リリースタグの削除、リポジトリ設定変更。

## 該当しない場合

PR 本文に「該当なし」と明記する。
